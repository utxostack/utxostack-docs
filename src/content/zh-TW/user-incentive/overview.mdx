# 概覽

歡迎參加 UTXO Stack [用戶激勵活動](https://www.utxostack.network/airdrop)！此活動旨在通過獎勵用戶積分，鼓勵用戶積極參與閃電網絡生態。

## 活動目標

- **用戶學習**：通過為用戶開通通道、進行交易和邀請他人提供激勵，促進閃電網絡的使用。
- **建立社區**：創建一個支持性的環境，讓用戶可以一起學習、分享和成長。
- **獎勵參與**：為積極參與的用戶提供積分和代幣獎勵。

請注意，此活動尚未向公眾開放。但您現在可以開始使用 JoyID 閃電錢包並為活動做好準備。

通過參與 UTXO Stack 用戶激勵活動，您不僅可以獲得獎勵，還可以為閃電網絡的增長和成功做出貢獻。我們很高興能與您一起踏上這段旅程！
