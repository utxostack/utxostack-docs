# 使用您的閃電錢包進行支付

## 關鍵概念

- **發票**：由接收方創建的付款請求。
- **入帳流動性**：可以通過支付通道接收的資金量。
- **出帳流動性**：您在通道中的餘額。可以通過支付通道發送的資金量。
- **通道容量**：兩個節點為通道提供的總金額。
- **儲備金**：由於某些安全原因，通道有儲備金。此金額不可用於發送。
- **閃電網絡費用**：閃電網絡節點為轉發交易收取的費用。

## 創建發票並接收

1. **開始接收**
   - 點擊"接收"
   - 選擇"Sat"
   ![快速選擇 Sat](/image/receive-1.gif)

2. **創建您的發票**
   - 輸入您想要的金額
   - 添加備註（可選但有幫助！）
   - 確認詳細信息
   ![設置支付](/image/receive-2.gif)

3. **分享並收款**
   - 分享您的發票或二維碼
   
   <Callout emoji="💡">
   保持螢幕活躍：確保您的 JoyID 帳戶和二維碼在收款前保持可見。
   </Callout>
   
   ![準備接收](/image/receive-3.gif)

## 通過閃電網絡發送

1. **開始發送**
   - 點擊"發送"
   - 選擇"Sats"
   ![開始支付](/image/send-1.gif)

2. **輸入支付詳情**
   - 粘貼發票或掃描二維碼
   ![快速輸入發票](/image/send-2.gif)

3. **完成支付**
   - 仔細檢查詳細信息
   - 確認並發送
   ![完成交易](/image/send-3.gif)
