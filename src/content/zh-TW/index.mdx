# 關於 UTXO Stack

## 什麼是 UTXO Stack？

[UTXO Stack](https://www.utxostack.network/) 是一個為混合閃電網絡（Hybrid LN）設計的**去中心化流動性質押層**，旨在解決閃電網絡（LN）採用過程中的關鍵挑戰，包括流動性不足、准入門檻高以及缺乏穩定幣支持等問題。

UTXO Stack 的完整基礎設施包括去中心化流動性質押池（DLSP）、混合閃電網絡（LN + [Fiber Network (FN)](https://github.com/nervosnetwork/fiber/blob/main/docs/light-paper.md)），以及 [JoyID 錢包](https://joy.id/en) 和 P2P 支付 SDK 等支持工具，為比特幣閃電生態系統提供可擴展、用戶友好且具互操作性的解決方案。

## UTXO Stack 的目標

1. **增強流動性**  
   UTXO Stack 通過其 DLSP 聚合和優化流動性，解決閃電網絡的流動性挑戰。

2. **支持穩定幣支付**  
   通過混合閃電網絡，UTXO Stack 將 [USDI](https://www.interpaystellar.com/) 等穩定幣引入閃電網絡，彌合比特幣波動性與日常實際支付之間的差距。

3. **降低准入門檻**  
   零成本通道開通（ZCCO）機制消除了開通閃電網絡通道的前期成本，簡化了新用戶的進入流程，擴大採用範圍。

4. **賦能閃電網絡**  
   UTXO Stack 致力於將閃電網絡轉變為**全球點對點支付基礎設施**，讓每個人都能使用無縫且去中心化的支付服務。

## 核心技術

1. **混合閃電網絡（LN + FN）**  
   - 將閃電網絡與 Fiber Network 整合，利用 Nervos CKB 的圖靈完備能力，實現無縫的資產互操作性和穩定幣支持。  
   - 促進比特幣和穩定幣之間的跨網絡流動性和原子交換。

2. **去中心化流動性質押池（DLSP）**  
   - 將 BTC、CKB 和穩定幣流動性聚合到去中心化資金池中。  
   - 為 LN 和 FN 節點提供流動性，實現高效路由和可靠支付。  
   - 引入流動性質押衍生品（LSD），為 DeFi 應用解鎖質押資產流動性。

3. **RGB++ 協議**  
   - [RGB++](https://github.com/ckb-cell/RGBPlusPlus-design) 是為比特幣區塊鏈設計的去中心化資產發行協議。 
   - 通過 RGB++，穩定幣可以在比特幣上發行並無縫流入混合閃電網絡。  

## UTXO Stack 如何賦能閃電網絡

1. **優化流動性**  
   通過解決流動性不足問題，UTXO Stack 確保閃電網絡上的交易更順暢，路由更高效。

2. **擴展使用場景**  
   通過穩定幣支持，LN 成為一個多功能生態系統，支持小額支付、跨境支付，甚至 AI 驅動的高頻交互。

3. **包容性採用**  
   零成本通道開通使 LN 更易接入，鼓勵個人用戶和企業參與比特幣閃電生態系統。
