# 사전 준비사항

UTXO Stack의 사용자 인센티브 이벤트에 참여하기 전에 필요한 설정 단계를 안내합니다:
- 초대 코드를 얻는 방법
- JoyID 라이트닝 지갑 생성을 위한 단계별 안내

이러한 사전 준비를 완료한 후, 지갑에 자금을 충전하고 결제를 통해 보상을 받으세요!
이미 JoyID 라이트닝 지갑을 보유하고 계신다면 이 문서를 건너뛰어도 됩니다.

## 1. 초대 코드 받기
현재 베타 테스트 단계에 있습니다. 플랫폼의 안정성을 보장하기 위해 초대 전용 시스템을 채택하여, 커뮤니티가 점진적으로 성장하면서 제품을 지속적으로 개선할 수 있도록 하고 있습니다.

초대 코드 받는 방법:
1. **텔레그램** [@UTXOStack](https://t.me/utxo_stack)에 가입하고, 초대 코드 구역 방문: [활동 세부사항과 최신 업데이트 받기](https://t.me/utxo_stack/13343)
2. [@UTXOStack](https://x.com/utxostack)을 **팔로우**하고, 파트너 프로젝트의 기회를 얻으세요

<Callout emoji="⚠️">
초대 코드에 대한 중요 사항:
- 각 코드는 한 번만 사용 가능합니다
- 현재 공개 출시 전까지 사용자는 새로운 초대 코드를 생성할 수 없습니다
</Callout>

## 2. 라이트닝 지갑 생성하기 ⏱️ (1-3분 소요)

### 시스템 요구사항
- **지원 브라우저**: Chrome, Edge, Safari
- **기기**: 
  - **안드로이드:** 버전 7.0 이상, Google Mobile Services 활성화 필요
  - **iOS:** iOS 16 이상, iCloud 서비스 활성화 필요
  - **macOS:** macOS Catalina 이상, Touch ID 지원 필요
  - **Windows:** TPM(신뢰할 수 있는 플랫폼 모듈) 필요
  - **Linux:** 지원됨

### 1단계: JoyID 지갑 생성하기

![create-joyid-wallet](/image/create-joyid-wallet.gif)

1. 브라우저(Chrome / Edge / Safari)로 [app.joy.id](https://app.joy.id/) 방문
2. "새 지갑 만들기" 클릭
3. 인증 과정 승인(2번의 인증 필요)
4. 축하합니다, 지갑이 성공적으로 생성되었습니다!

<Callout emoji="💡">
지갑 생성에 실패하거나 문제가 발생한 경우:

- [FAQ](https://www.notion.so/ca5b5cd04d7e4fd3899ce6c65b334fcc?pvs=21) 문서 확인
- [JoyID Discord](https://discord.com/invite/77MyakRKVB)에 가입하여 관리자에게 도움 요청
</Callout>

### 2단계: 라이트닝 네트워크로 전환하기

![switch-to-lightning-network](/image/switch-network.gif)

1. 우측 상단의 네트워크 아이콘(**₿**)을 클릭
2. "Lightning (Layer2)" 네트워크 선택
3. "초대 코드" 입력
    
    <Callout emoji="💡">
    팁: 초대 QR 코드를 스캔할 때는 JoyID 스캔 대신 기기의 카메라를 사용하는 것이 좋습니다.
    </Callout>
    
4. 인증 과정 완료
5. 축하합니다, 라이트닝 지갑이 성공적으로 설정되었습니다!

## 문제 해결 FAQ
일반적인 문제와 해결방법:
1. **지갑 생성 실패**
   - 브라우저 캐시를 지우고 다시 시도
   - iCloud/Google 계정이 로그인되어 있는지 확인
   - 기기가 시스템 요구사항을 충족하는지 확인

2. **초대 코드 문제**
   - 코드가 만료되지 않았는지 확인
   - 올바른 입력 형식인지 확인
   - 코드가 유효해 보이지만 작동하지 않는 경우 지원팀에 문의
