# 概述

欢迎参加 UTXO Stack [用户激励活动](https://www.utxostack.network/airdrop)！此活动旨在通过奖励用户积分，鼓励用户积极参与闪电网络生态。

## 活动目标

- **用户学习**：通过为用户开通通道、进行交易和邀请他人提供激励，促进闪电网络的使用。
- **建立社区**：创建一个支持性的环境，让用户可以一起学习、分享和成长。
- **奖励参与**：为积极参与的用户提供积分和代币奖励。

请注意，此活动尚未向公众开放。但您现在可以开始使用 JoyID 闪电钱包并为活动做好准备。

通过参与 UTXO Stack 用户激励活动，您不仅可以获得奖励，还可以为闪电网络的增长和成功做出贡献。我们很高兴能与您一起踏上这段旅程！
