# 概述

欢迎参加 UTXO Stack [用户激励活动](https://www.utxostack.network/airdrop)！此活动旨在通过奖励用户积分，鼓励用户积极参与闪电网络生态。

## 活动目标

- **用户学习**：通过为用户开通通道、进行交易和邀请他人提供激励，促进闪电网络的使用。
- **建立社区**：创建一个支持性的环境，让用户可以一起学习、分享和成长。
- **奖励参与**：为积极参与的用户提供积分和代币奖励。

请注意，此活动尚未向公众开放。但您现在可以开始使用 JoyID 闪电钱包并为活动做好准备。

## 如何赚取积分

用户可以通过完成闪电网络生态系统中的各种任务来赚取积分。以下是开始的方法：

1. **开通闪电网络通道**：建立一个新通道以开始在网络上进行交易。[了解如何开通并充值通道](./open-and-fund-your-channel.md)
2. **进行支付**：使用您的通道发送和接收支付，展示快速、低成本交易的力量。[查看支付指南](./make-payments.md)
3. **参与特别活动**：留意提供额外积分赚取机会的特别活动和促销。

## 如何查看您的积分

要检查您当前的积分余额并跟踪您的进度：

1. **登录您的账户**：通过 [UTXO Stack 平台](https://www.utxostack.network/)访问您的用户仪表板。
2. **导航到积分部分**：找到专门用于您的积分余额和最近活动的部分。
3. **查看您的活动**：查看您如何赚取积分的详细信息以及有哪些任务可提供更多机会。
