# 前置条件

本指南介绍参与 UTXO Stack 用户激励活动前需要完成的设置步骤：
- 获取邀请码
- 创建 JoyID 闪电钱包

完成这些前置条件后，您就可以为钱包充值并进行了！
如果您已经有 JoyID 闪电钱包，可以跳过本文档。

## 1. 获取邀请码
我们目前处于 Beta 测试阶段。为确保平台稳定性，我们采用了邀请制，让社区能够逐步增长，同时帮助我们持续改进产品。

获取邀请码的方法：
1. **加入 TG** [@UTXOStack](https://t.me/utxo_stack)，访问邀请码专区：[获取活动详情和最新更新](https://t.me/utxo_stack/13343)
2. **关注** [@UTXOStack](https://x.com/utxostack)，获取合作项目机会

<Callout emoji="⚠️">
关于邀请码的重要说明：
- 每个邀请码只能使用一次
- 在公开发布前，用户无法生成新的邀请码
</Callout>

## 2. 创建闪电钱包 ⏱️ (1-3分钟)

### 系统要求
- **支持的浏览器**: Chrome、Edge、Safari
- **设备**: 
  - **安卓:** 需要 7.0 或更高版本，并启用 Google Mobile Services
  - **iOS:** 需要 iOS 16 或更高版本，并启用 iCloud 服务
  - **macOS:** 需要 macOS Catalina 或更高版本，支持 Touch ID
  - **Windows:** 需要 TPM (可信平台模块)
  - **Linux:** 支持

### 第一步：创建 JoyID 钱包

![create-joyid-wallet](/image/create-joyid-wallet.gif)

1. 通过浏览器（Chrome / Edge / Safari）访问 [app.joy.id](https://app.joy.id/)
2. 点击"创建新钱包"
3. 授权验证过程（需要两次授权）
4. 恭喜，您的钱包已成功创建！

<Callout emoji="💡">
如果创建失败或遇到问题：

- 查看 [FAQ](https://www.notion.so/ca5b5cd04d7e4fd3899ce6c65b334fcc?pvs=21) 文档
- 加入 [JoyID Discord](https://discord.com/invite/77MyakRKVB) 并向管理员寻求帮助
</Callout>

### 第二步：切换到闪电网络

![switch-to-lightning-network](/image/switch-network.gif)

1. 点击右上角的网络图标（如 **₿**）
2. 选择"Lightning (Layer2)"网络
3. 输入"邀请码"
    
    <Callout emoji="💡">
    提示：如果您正在扫描邀请码二维码，建议使用设备的相机而不是 JoyID 扫描。
    </Callout>
    
4. 完成授权过程
5. 恭喜，您的闪电钱包已成功设置！

## 问题解决 FAQ
常见问题和解决方法：
1. **钱包创建失败**
   - 清除浏览器缓存并重试
   - 确保您的 iCloud/Google 账户已登录
   - 检查您的设备是否符合系统要求

2. **邀请码问题**
   - 验证代码是否已过期
   - 确保输入格式正确
   - 如果邀请码有效但无法使用，请到 [JoyID Discord](https://discord.com/invite/77MyakRKVB) 联系我们
